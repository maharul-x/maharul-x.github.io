## 🛠 Built With

- **HTML5** — Semantic structure
- **CSS3** — Custom properties, animations, responsive grid layout

---

## ✨ Features

- 🎨 Dark theme with a clean, modern design
- 📱 Fully responsive — works on mobile, tablet & desktop
- ⚡ Smooth scroll animations (IntersectionObserver)
- 🧭 Active navigation highlight on scroll
- 📂 Sections: Hero, About, Experience, Education, Projects, Skills, Extras, Contact
- 🚀 Zero dependencies — no frameworks, no libraries

## 🚀 Getting Started Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/maharul-x/maharul-x.github.io.git
   ```

2. **Open the folder**
   ```bash
   cd maharul-x.github.io
   ```

3. **Run it in your browser**
   - Simply open `index.html` in any browser, **or**
   - Use the **Live Server** extension in VS Code for live reload
